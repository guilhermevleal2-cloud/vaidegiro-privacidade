Política de Privacidade - Vai de Giro Motorista

Última atualização: Maio 2026

O aplicativo Vai de Giro Motorista respeita a privacidade dos usuários e explica nesta política como os dados são coletados, utilizados e protegidos.

Dados coletados

  • Nome
  • Número de telefone
  • Email
  • Localização em tempo real
  • Informações do dispositivo
  • Dados necessários para funcionamento das corridas

Uso dos dados

  • Conectar passageiros e motoristas
  • Permitir rastreamento das viagens
  • Melhorar a experiência do usuário
  • Garantir segurança da plataforma

Compartilhamento

Os dados não são vendidos a terceiros.
